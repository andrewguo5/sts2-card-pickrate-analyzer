# Card Pick Rate Visualization

Interactive React-based web application for visualizing STS2 card pick rates by floor.

## Features

- **Interactive Card Selection**: Browse and search through all cards
- **Beautiful Charts**: Recharts-powered area charts showing smoothed pick rates
- **Detailed Data Table**: Floor-by-floor breakdown with visual progress bars
- **Responsive Design**: Works on desktop and mobile
- **Real-time Filtering**: Search cards by name
- **Dual Visualization**: See both raw and kernel-smoothed pick rates

## How to Use

### Option 1: Open Directly in Browser

1. Make sure `card_pickrates.json` is in the same directory as `index.html`
2. Open `index.html` in your web browser
3. That's it! No build step required.

### Option 2: Use a Local Server (Recommended)

For better performance and to avoid CORS issues:

```bash
# Using Python 3
python3 -m http.server 8000

# Then open: http://localhost:8000
```

Or with any other local server tool.

## Generating Fresh Data

To update the visualization with new run data:

```bash
# 1. Sync latest runs from game
../sync_runs.sh

# 2. Generate new pick rate data
python3 ../card_pickrate_analysis.py \
    --character CHARACTER.REGENT \
    --ascension A10 \
    --bandwidth 2 \
    --output card_pickrates.json

# 3. Refresh your browser
```

## What You'll See

### Sidebar (Left)
- Search box to filter cards
- List of all cards sorted by total offers
- Each card shows: name, offer count, overall pick rate
- Click any card to view details

### Main Panel (Right)
- **Summary Stats**: Total offered, picked, overall rate, floors seen
- **Interactive Chart**:
  - Blue area: Kernel-smoothed pick rate (b=2)
  - Purple dashed line: Raw pick rate
  - Hover for exact values
- **Data Table**: Complete floor-by-floor breakdown

## Customization

You can modify the data source by running the analysis with different parameters:

```bash
# Different character
python3 ../card_pickrate_analysis.py --character CHARACTER.IRONCLAD --ascension A10

# Different ascension levels
python3 ../card_pickrate_analysis.py --character CHARACTER.REGENT --ascension A0-9

# Different bandwidth
python3 ../card_pickrate_analysis.py --character CHARACTER.REGENT --ascension A10 --bandwidth 3
```

## Technical Details

- **Framework**: React 18 (loaded via CDN, no build step)
- **Charts**: Recharts 2.5
- **Data Format**: JSON export from card_pickrate_analysis.py
- **Styling**: Pure CSS, no frameworks
- **Browser Support**: Modern browsers (Chrome, Firefox, Safari, Edge)
